Class RegisterDto

Clase que representa un objeto de transferencia de datos (DTO) para el registro de un usuario.

Esta clase se utiliza para transferir la información necesaria para registrar un nuevo usuario en el sistema, incluyendo el nombre, correo electrónico, fecha de nacimiento, contraseña y rol. Los datos son encapsulados en la clase, proporcionando getters y setters para acceder y modificar los valores.
